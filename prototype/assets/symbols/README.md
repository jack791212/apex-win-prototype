# 老虎機符號素材（暗影儀式 Shadow Ritual）

把符號圖檔放在此資料夾，並將 `src/views/slot.js` 最上方的 `ART_ENABLED` 改為 `true`，
老虎機就會自動改用圖片；找不到對應圖檔時會回退為內建 emoji。

## 檔名規則（PNG，建議去背、正方形比例）

- 低分：`L1.png` ~ `L5.png`
- 中分：`M1.png` ~ `M5.png`
- 高分：`H1.png` ~ `H5.png`
- Wild：`W.png`
- Scatter（愛心）：`S.png`

賠率對應請見 `02 規格`／付費表（L=低、M=中、H=高，數字越小賠率越高）。

## ⚠️ 著作權

請只放入「**你自有 / 已取得授權 / 委外原創**」的美術素材。
**請勿放入未授權的他人商業遊戲（如市售 slot）之圖檔**並公開發佈，
否則屬侵權。本系統僅提供「替換機制」，素材合法性由放置者負責。
