## 特色表

### 遊戲 / 內容
- 娛樂城 1,500+ 款（slot + 真人荷官）。真人荷官 = 供應商接入/真金 → **avoid**。
- 無 crypto Originals 文化（與 Stake/Roobet 不同）；強項是體育 in-play。

### 留存 / 促銷
- **每日 Bet Boosts**：首頁醒目位置每日推「加倍賠率」精選盤口 → 純前端可借：每日精選遊戲/加成輪播。
- Bet Builder+：單張最多 12 個選項、跨 14 種運動、支援盤中組合 → 體育向，**avoid**（需牌照真金 + 賽事數據源）。

### UX / 上手（最可學的部分）
- **首頁資訊架構**：把「當下最熱市場」與「每日加成/優惠」清楚置頂，版面簡潔，主打「slick & speedy」快速服務。
- **深色 UI + 綠白點綴**，導覽分區乾淨：Home / Live / Sports / Account 四大清楚分流。
- **In-Play 即時數據面板**：盤中下注區附詳細統計與即時比分（資訊密度高但不雜亂）。

### 金流 / 模式（只記錄，不推進）
- 真金體育博彩 + 真人荷官 + 直播串流 → 全屬需牌照 / 供應商接入 = **CONTROL.avoid**。

## 附錄：2026-07-03 刷新

本輪（T1 每 7 天到期）刷新，**主體無變化**（核心仍是真金體育+真人荷官＝avoid）。新記三個純前端可學的 nuance：

1. **Bet-slip 即時改價**：下注前直接在注單上更新價格/賠率，讓玩家「送出前先看到最終數字」＝低意外感的確認 UX。→ ApexWin 互動回合/下注確認可借：**送出前即時預覽最終賠付/成本**（複用既有下注流，S）。
2. **累進式加成（parlay/leg 越多加成越高，最高 +100%）**：加成幅度隨組合複雜度遞增＝獎勵「更投入的行為」。→ 可移植為**連對/連關卡越深、加成％越高**的純前端軟加成呈現（呼應 #27 Hilo 連對累乘，S）。
3. **設計決策：維持固定核心優惠，不像對手每週輪替 promo**。→ 與 Courtside（無 VIP、靠輪替 contests）、WOW Vegas（衰減 VIP）並列＝**第 N 種留存哲學記錄**：「穩定可預期的常設優惠」vs「不斷輪替製造新鮮感」。ApexWin 目前偏常設，記錄兩種取向供權衡，非缺口。

> 結論：bet365 仍是「UX 教材、非機制來源」。無新增缺口卡建議，既有 3 點子（每日精選輪播／partial cash-out／熱度牆）維持。下次到期 2026-07-10。
