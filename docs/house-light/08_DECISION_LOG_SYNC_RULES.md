## 本機定位

本機只保存決策同步規則，不保存正式決策資料庫。

## 需要記錄的決策

以下內容必須記錄：

- 玩法方向改變
- UX Flow 改變
- MVP 範圍改變
- Prototype 技術方向改變
- 任務優先級重大調整
- 高層要求
- 使用者確認的產品方向
- Claude 實作造成的規格差異
- Codex 驗收發現的重要問題

- 決策 ID
- 日期
- 決策狀態
- 決策來源
- 背景
- 決策內容
- 影響範圍
- 後續任務
- Notion 同步狀態
